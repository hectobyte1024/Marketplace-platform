# Frontend

React + TypeScript frontend application for the marketplace platform.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Visit http://localhost:3001

## Build

```bash
pnpm build
```

## Features

- 🏠 Home page with feature overview
- 📍 Browse and search workspaces
- 📅 Booking management
- 🔍 Location-based filtering
- 💰 Real-time pricing
